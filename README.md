# GRIDPLAN
A fun game using GPLAN

## How to Run?

1. Install libraries from requirements.txt.

**Windows**
```bash
pip install -r requirements.txt
```
**Unix/macOS**
```bash
python3 -m pip install -r requirements.txt
```

2. Execute the main.py file.

**Windows**
```bash
py main.py
```
**Unix/macOS**
```bash
python3 main.py
```
